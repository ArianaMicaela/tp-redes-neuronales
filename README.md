### **Trabajo Práctico: Redes Neuronales**



###### **Descripción**



Este repositorio contiene el desarrollo de un trabajo práctico enfocado en la implementación de una red neuronal utilizando Python.



El objetivo es comprender el funcionamiento interno de los modelos de aprendizaje incluyendo entrenamiento, evaluación y análisis de resultados.





###### **Tecnologías utilizadas**



* Python
* NumPy
* Pandas





###### **Contenido**



* PazAriana\_TP.ipynb → Implementación del modelo
* PazAriana\_TP.pdf → Reporte con análisis y conclusiones





###### **Desarrollo**



Se implementa una red neuronal desde cero, incluyendo:



* Inicialización de parámetros
* Forward propagation
* Cálculo de la función de pérdida
* Backpropagation
* Actualización de pesos





###### **Resultados**



Se evalúa el rendimiento del modelo sobre datos de entrenamiento y test, analizando su capacidad de generalización.
